# nie Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a single-host Linux daemon that enforces default-deny IPv4 egress based on hostnames learned through a local DNS proxy, with `enforce` and `audit` modes.

**Architecture:** The userspace daemon owns config parsing, glob policy evaluation, DNS forwarding, trust-entry learning, redirect-rule management, and eBPF lifecycle. A tc egress classifier on one interface reads a pinned IPv4 allow map, bypasses proxy-marked packets, and either drops or audit-logs unknown destinations. The implementation stays IPv4-only and supports classic DNS over UDP/TCP port 53 in v1.

**Tech Stack:** Go, `github.com/miekg/dns`, `github.com/cilium/ebpf`, `gopkg.in/yaml.v3`, `golang.org/x/sys/unix`, tc/eBPF, iptables or nftables shell integration

```go
type TrustEntry struct {
	IPv4      netip.Addr
	ExpiresAt time.Time
}

func NewEntry(rawIP string, ttl uint32, now time.Time, maxTTL time.Duration) (TrustEntry, error) {
	ip, err := netip.ParseAddr(rawIP)
	if err != nil || !ip.Is4() {
		return TrustEntry{}, fmt.Errorf("invalid IPv4: %q", rawIP)
	}
	d := time.Duration(ttl) * time.Second
	if d <= 0 || d > maxTTL {
		d = maxTTL
	}
	return TrustEntry{IPv4: ip, ExpiresAt: now.Add(d)}, nil
}
```

Also define the small interface that the DNS proxy will depend on:

```go
type TrustWriter interface {
```

## Execution Notes

- Do not start with the tc program. The safe order is config -> policy -> trust types -> DNS proxy -> redirect manager -> userspace eBPF loader -> tc program -> runtime wiring -> smoke tests.
- Keep `internal/dnsproxy` dependent only on the policy interface and the small `TrustWriter` interface. Do not make it depend directly on concrete eBPF loader types.
- Avoid putting iptables execution directly in `cmd/nie/main.go`; keep all shelling out inside `internal/redirect`.
- When the eBPF value format changes, update `bpf/include/common.h`, Go encoder/decoder code, and related tests in the same commit.
